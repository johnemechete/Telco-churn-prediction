# Telco Customer Churn Analysis

This project aims to predict customer churn for a telecom company using machine learning.

## Project Status
- ✅ Exploratory Data Analysis (EDA) completed
- ⚪ Predictive modeling in progress

## EDA Highlights
- Data cleaning and preprocessing
- Insights from visualizations
- Feature correlations and distributions

## Getting Started
1. Clone the repo
2. Install dependencies from `requirements.txt`
3. Run `notebooks/eda.ipynb` to explore the dataset
